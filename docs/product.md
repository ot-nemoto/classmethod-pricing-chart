# product.md

## 目的

Classmethod 社の AWS 請求 CSV レポートをブラウザだけで可視化し、サービス別・アカウント別のコスト構造を素早く把握できるようにする。
インストール不要・サーバーレスで動作するため、AWS コンソールのダウンロードデータをそのまま分析できる。

## 対象ユーザー

- Classmethod 社内で複数 AWS アカウントのコストを管理・分析する担当者
- 月次の AWS 請求レポートをもとに、サービス別・アカウント別のコスト傾向を把握したいエンジニアや管理者

## 成功指標

- CSV をアップロードしてから 3 秒以内にチャートが描画される
- 複数月・複数アカウントのデータを 1 画面で比較できる
- フィルタリングや集計モードの切り替えが直感的に操作できる
