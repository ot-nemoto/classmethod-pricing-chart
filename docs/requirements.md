# requirements.md

## 機能要件

### CSV インポート

- `monthly-report-YYYY-MM-ACCOUNTID.csv` 形式のファイルをドラッグ＆ドロップまたはファイル選択で取り込む
- 複数ファイルを同時にアップロードできる
- 同一アカウント・同一月のファイルを再アップロードすると上書き更新される
- ファイル名から年月（YYYY-MM）と AWS アカウント ID を自動抽出する
- 必須カラム: `product_name`、`cost`（`$` や `,` を含む文字列も正規化して数値変換）
- 必須カラムが欠損している場合は警告を表示し、そのファイルのデータは取り込まない
- パースエラーは画面上に表示し、警告は最大 5 件まで表示する

### フィルタリング

- アカウント・年月・サービスをチェックボックスで絞り込める
- 各フィルターにテキスト検索欄を持つ
- サービスフィルターに「Top10」一括選択ボタンを持つ
- 全選択・全解除ボタンを各フィルターに持つ

### 集計・チャート表示

- フィルター条件に応じた積み上げ棒グラフをリアルタイムで更新する
- 集計モードを「サービス別」「アカウント別」で切り替えられる
- 時間軸を「月次」「年次」で切り替えられる
- 選択条件の合計金額を USD でリアルタイム表示する

### データ管理

- 「クリア」ボタンで全読み込みデータを初期化できる
- 追加アップロードで既存データに差分追加できる

## 非機能要件

- サーバーレス・静的ホスティング（GitHub Pages）で動作する
- すべての処理はブラウザ内で完結し、データを外部に送信しない
- モダンブラウザ（Chrome / Edge / Firefox 最新版）で動作する
- モバイル対応は必須としない（PC ブラウザを前提とする）
